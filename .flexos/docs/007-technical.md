# Technical

## Stack
| Layer | Choice | Rationale |
|-------|--------|-----------|
| Framework | Nuxt 4 (Vue) | Fast, modern, builds a beautiful PWA that feels native without App Store delays. |
| Styling | CSS Modules / Tokens | Native CSS with FlexOS token system for predictable theming. |
| Database | Supabase (PostgreSQL) | Relational data for economy + real-time sync for points and comments. |
| Auth | Supabase Auth | Frictionless social login (Google/Apple) to prevent drop-off. |
| AI | Anthropic Claude 3.5 Sonnet | Superior at parsing cultural context, nuance, and humor compared to standard translation models. |
| Hosting | Vercel | Edge network delivery ensures images and API routes load instantly globally. |
| Storage | Vercel Blob | Handles massive volume of UGC meme images with on-the-fly optimization. |

## Architecture
- **Rendering Strategy:** SPA mode for the authenticated app (`/feed`, `/vault`) to ensure zero-latency swiping and persistent audio/haptics. SSR for the public landing page (`/`) for SEO and fast initial load.
- **Data Flow:** Browser → Nuxt Server API (`/api/*`) → Supabase / Anthropic. The browser never calls Supabase or Anthropic directly to protect secrets and enforce tier limits.

## Key Configuration
- Nuxt configured for PWA (manifest, service worker for caching images).
- FlexOS conventions: explicit component registration, no auto-imports for critical state.

## Environment Variables
- `NUXT_PUBLIC_SUPABASE_URL` (Public) - Supabase endpoint.
- `NUXT_PUBLIC_SUPABASE_ANON_KEY` (Public) - Supabase anon key.
- `SUPABASE_SERVICE_ROLE_KEY` (Server) - Admin overrides.
- `ANTHROPIC_API_KEY` (Server) - Claude API access.
- `BLOB_READ_WRITE_TOKEN` (Server) - Vercel Blob access.

## Auth
- Supabase Auth with OAuth providers.
- Sessions stored in httpOnly cookies (managed by Nuxt server routes) for security.
- 30-day expiry.

## API Design
- `GET /api/memes/feed` - Returns batch of 5 memes, cursor-based pagination.
- `POST /api/attempts` - Submits a guess, calculates points server-side, returns updated balance and breakdown.
- `POST /api/ugc/analyze` - Accepts image upload, returns Claude 3.5 analysis.

## Key Integrations
- **Anthropic:** Called during Field Notes upload. If it fails, return graceful error "AI is resting".
- **Vercel Blob:** Direct upload from client using short-lived tokens generated by server.

## Security Checklist
- Input validation on all meme guesses (prevent XSS in comments).
- Rate limiting on `/api/attempts` to prevent botting points.
- Rate limiting on `/api/ugc/analyze` (expensive AI call).
- RLS policies in Supabase to ensure users can only spend their own points.

## Performance Targets
- LCP < 1.5s on landing page.
- TTFB < 200ms for `/api/memes/feed`.
- Images must load instantly during swipe (pre-fetch next 3 images in client).

## Testing Strategy
- Unit tests for points calculation logic.
- E2E tests for the core Feed loop (swipe, guess, breakdown).
- Integration tests for Anthropic API prompt parsing.
