# Flows

## Core Flows

### 1. Translate & Understand (Core Loop)
**Involves:** Learner, System
1. User views meme image and interaction zone in The Feed.
2. User enters guess or selects option.
   - *Branch A (Correct):* 
     - Screen shakes, haptic buzz.
     - Input turns toxic neon green.
     - Points calculated (Base 50 + Speed Bonus + Streak Multiplier).
     - Points visually fly to top bar.
     - `attempts` record created.
   - *Branch B (Incorrect/Skip):* 
     - Red shake animation, dull haptic thud.
     - Correct answer revealed in muted color.
     - 0 points awarded.
     - `attempts` record created.
3. Bottom sheet slides up displaying AI-generated Culture Breakdown.
4. User reads breakdown.
5. User optionally taps "Save to Deck" → creates `saves` record with default SRS intervals.
6. User swipes up → next meme snaps into view.

### 2. The Vault Flex (Economy Loop)
**Involves:** Learner, System
1. User navigates to The Vault.
2. User taps a locked emoji.
   - *Branch A (Insufficient points):* Button disabled, snarky tooltip ("Keep scrolling, you're broke").
   - *Branch B (Sufficient points, Premium Item, Free User):* Prompts Pro Tier upgrade modal.
   - *Branch C (Sufficient points, valid tier):* User confirms purchase.
3. System calls secure RPC to deduct points and create `Unlock` record atomically.
4. Arcade-style unlock animation plays (particles, silhouette flash).
5. User navigates to a meme's comment section and uses the unlocked sticker.

### 3. Field Note Translation (UGC Pipeline)
**Involves:** Learner, AI System
1. User uploads a screenshot to Field Notes.
2. Loading state: "Decoding the culture..."
3. System calls Anthropic Claude 3.5 Sonnet API with image and system prompt.
4. AI extracts text, generates breakdown, and formats JSON response.
5. User reviews the breakdown and learns the joke.
6. App prompts: "Publish to Community for royalties?"
   - *Branch A (Yes):* Meme `status` set to `pending_review` (or `published` if trusted user). User gets passive points when others solve it.
   - *Branch B (No):* Saved privately to user's Deck.

### 4. Spaced Repetition Review (The Deck)
**Involves:** Learner, System
1. User opens The Deck and taps "Review Due Cards".
2. System fetches `saves` where `nextReviewDate <= today`.
3. User views meme image (slang obscured).
4. User attempts to recall slang, taps to reveal answer and breakdown.
5. User self-grades recall: "Hard", "Good", or "Easy".
6. System calculates new `interval` and `easeFactor` (SuperMemo-2), updates `saves` record.
7. Next card loads.

### 5. Community Moderation (Flagging)
**Involves:** Learner, System
1. User spots an offensive or incorrectly translated meme in The Feed.
2. User taps "Flag" icon in action bar.
3. User selects reason (e.g., "Incorrect Translation").
4. System creates `reports` record.
5. If a meme receives > 3 reports, system updates meme `status` to `flagged`, removing it from active feed queries until admin review.

## State Machines

`pending_ai_processing` → `ai_completed` → `awaiting_user_publish` → `published` → `flagged` (if reported) → `removed` (if admin confirms) or `published` (if admin clears).

## Error States
- **AI Processing Failure (Field Notes):** If Anthropic API times out or refuses content (safety filter), show "The meme was too dank for our AI. Try another." Allow retry.
- **Network Disconnect during Feed:** Cache next 5 memes locally. If cache empty, show "You've reached the end of the internet. Reconnect to load more."
- **Double Spend in Vault:** Prevented by PostgreSQL Trigger. Show "Transaction failed. Are you trying to hack the matrix?"
