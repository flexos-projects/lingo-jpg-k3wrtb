### 3. Field Note Translation (UGC)
**Involves:** Learner, AI System
1. User uploads a screenshot to Field Notes.
2. Loading state: "Decoding the culture..."
3. AI runs OCR, extracts text, generates breakdown.
4. User reviews the breakdown and learns the joke.
5. App prompts: "Turn this into a challenge for the community?"
   - *Branch A (Yes):* Meme added to global feed. User gets passive points when others solve it.
   - *Branch B (No):* Saved privately to user's Deck.
