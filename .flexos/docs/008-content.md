# Content

The cold-start problem is the biggest hurdle. Lingo.jpg cannot launch with an empty feed. It needs a meticulously curated database of "Seed Memes" across 2-3 starting languages (e.g., Spanish, Japanese, French). These memes need to be hand-picked for their educational value—they should teach common, highly useful slang that textbooks ignore.

**What we need to create:**

1.  **The Seed Curriculum:** 500 hand-curated memes per launch language. These need OCR extraction, literal translations, and AI-generated (but human-reviewed) Culture Breakdowns. 
2.  **The Vault Assets:** The initial batch of unlockable rewards. We need 20-30 custom, high-quality digital stickers/emojis per language. These must reference internet culture (e.g., a "This is Fine" dog wearing a sombrero, or a Pepe the Frog bowing).
3.  **The AI Prompts:** The system instructions for the Culture Breakdown AI. It needs to be trained to avoid academic language, keep explanations under 50 words, and perfectly capture the tone of the joke.
4.  **Marketing Copy:** The landing page needs to hit hard. "Stop learning how to order a train ticket. Start learning how to understand the group chat."
