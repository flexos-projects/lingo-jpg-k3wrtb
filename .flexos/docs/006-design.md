# Design

## Design Direction
Lingo.jpg does not look like a classroom. It looks like the internet.
The aesthetic is heavily inspired by late-night scrolling. **Dark mode is the default and only mode.** The background is a deep, bruised purple-black, allowing the bright, low-res memes to pop. Accents are pure cyber-neon: toxic radioactive green for correct answers and active streaks, vibrant electric purple for The Vault and premium interactions. 
It channels "TikTok's dark mode UI meets Discord's custom emoji culture." It explicitly avoids the sterile, flat, gamified look of Duolingo. It feels like a secret club for polyglots run by hackers.

## UX Patterns
- **The Infinite Scroll:** Like TikTok, the feed is full-screen and vertical. Swiping up is the primary navigation mechanism to move to the next challenge.
- **The Culture Reveal:** The breakdown doesn't load on a new page. It slides up as a bottom sheet over the meme, keeping the user in context. It feels snappy and immediate.
- **Haptic Validation:** When a user gets a translation right, the button changes to neon green, the phone buzzes aggressively, and points physically fly up the screen.

## Component Patterns
- **Meme Challenge Card:** Edge-to-edge image. Text overlay uses an impact-style font with heavy strokes to ensure readability over any image. The "blank" is a glowing underline that fills with text when guessed.
- **Vault Item Button:** Chunky, pill-shaped buttons with ambient neon shadows. Locked items are dark silhouettes.
- **Streak Counter:** Always visible in the app shell, using a slightly chaotic, marker-style display font.

## Layout Strategy
- **Mobile-First App Shell:** Bottom navigation bar with 5 icons (Feed, Deck, Upload, Vault, Profile). The Feed hides the nav bar while scrolling, revealing it on slight scroll up.
- Desktop is supported but framed like a mobile app (centered max-width container with blurred background), prioritizing the vertical feed experience.

## Motion & Animation Language
- **The Aha Moment:** Fast, punchy animations. A correct guess triggers a slight screen shake, a neon flash, and a satisfying *ding*.
- **The Unlock:** Arcade-style celebration. When buying an emoji, particles explode from the button, and the silhouette flashes white before revealing the colored asset.
- **Swiping:** Physics-based, smooth vertical snapping between memes.

## Microcopy Tone
Chronically online, irreverent, and snarky. It explains jokes without ruining them.
- *Correct Answer:* "Nailed it." or "No cap."
- *Wrong Answer:* "Oof. Try again."
- *Points Earned:* "+50 Glot Points. Don't spend it all in one place."
- *Vault Purchase:* "You are now the proud owner of Chilaquiles Reaction."

## Empty States
Empty states are snarky and push the user to action.
- **Deck Empty:** "Your deck is as empty as a textbook. Go scroll."
- **Profile Stats Empty:** "You haven't translated anything yet. We all start somewhere."

## Colour Direction
Deep purple-black backgrounds with high-contrast neon accents. Primary is toxic green (success, action), accent is electric purple (premium, vault).
