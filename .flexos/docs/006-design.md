# Design

## Design Direction
Lingo.jpg does not look like a classroom. It looks like the internet.
The aesthetic is heavily inspired by late-night scrolling and hacker culture. **Dark mode is the default and only mode.** The background is a deep, bruised purple-black, allowing the bright, low-res memes to pop. Accents are pure cyber-neon: toxic radioactive green for correct answers and active streaks, vibrant electric purple for The Vault and premium interactions. 
It channels "TikTok's dark mode UI meets Discord's custom emoji culture." It explicitly avoids the sterile, flat, corporate-memphis gamification of Duolingo. It feels like a secret club for polyglots run by hackers.

## UX Patterns
- **The Infinite Scroll:** Like TikTok, the feed is full-screen and vertical. Swiping up is the primary navigation mechanism to move to the next challenge. There are no "Next" buttons.
- **The Culture Reveal:** The breakdown doesn't load on a new page. It slides up as a bottom sheet over the meme, keeping the user in visual context of the joke. It feels snappy, immediate, and dismissible with a downward swipe.
- **Haptic & Visual Validation:** When a user gets a translation right, the input field changes to neon green, the phone buzzes aggressively, and points physically fly up the screen from the input to the top-bar counter. It must feel tactile and rewarding.

## Component Patterns
- **Meme Challenge Card:** Edge-to-edge image container. The text overlay uses an impact-style font with heavy black strokes to ensure readability over any chaotic meme background. The "blank" is a glowing underline that fills with text when guessed.
- **Streak Flame:** A chaotic, marker-style display component. When active, it pulses slightly. When at risk (under 4 hours), it turns a warning orange and shakes periodically.
- **Vault Item Button:** Chunky, pill-shaped buttons with ambient neon drop-shadows. Locked items are dark silhouettes. Premium items have a subtle animated iridescent border.
- **Scrambled Word Pills:** Draggable, physical-feeling blocks. When dropped in the correct order, they snap together with a satisfying magnetic click.

## Layout Strategy
- **Mobile-First App Shell:** Bottom navigation bar with 5 icons (Feed, Deck, Upload, Vault, Profile). The Feed hides the nav bar while scrolling down to maximize screen real estate, revealing it on a slight scroll up.
- **Desktop Adaptation:** Desktop is supported but framed like a mobile app (centered max-width container with a heavily blurred, ambient-colored background matching the current meme), prioritizing the vertical feed experience.

## Motion & Animation Language
- **The Aha Moment:** Fast, punchy animations. A correct guess triggers a slight screen shake (0.2s duration), a neon flash, and a satisfying *ding*.
- **The Vault Unlock:** Arcade-style celebration. When buying an emoji, particles explode from the button, the silhouette flashes pure white, and then reveals the colored asset with a slight bounce effect.
- **Swiping:** Physics-based, smooth vertical snapping between memes. Must feel completely frictionless.

## Microcopy Tone
Chronically online, irreverent, snarky, but highly educational. It explains jokes without ruining them.
- *Correct Answer:* "Nailed it." or "No cap."
- *Wrong Answer:* "Oof. Try again."
- *Points Earned:* "+50 Glot Points. Don't spend it all in one place."
- *Vault Purchase:* "You are now the proud owner of Chilaquiles Reaction."
- *Deck Review:* "Time to see if you actually remember anything."

## Empty States
Empty states are snarky and push the user to action.
- **Deck Empty:** "Your deck is as empty as a textbook. Go scroll."
- **Profile Stats Empty:** "You haven't translated anything yet. We all start somewhere."
- **Feed Cache Empty:** "You've reached the end of the internet. Reconnect to load more."

## Colour Direction
Deep purple-black backgrounds with high-contrast neon accents. Primary is toxic green (success, action, streaks), accent is electric purple (premium, vault, UGC royalties).
