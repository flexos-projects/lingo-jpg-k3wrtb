## Position/Computed Fields
- `pointsBalance`: Derived conceptually from `sum(attempts.pointsEarned) - sum(emojis.cost)`, but stored denormalised on the User record for fast reads. Updates via DB trigger or transaction on new Attempt/Unlock.

## Tier Enforcement
- `isPremium` emojis in The Vault can only be unlocked if `users.tier == 'pro'`. Enforced via Supabase RLS policies.
- Field Notes uploads count checked against a daily limit for free users.

## File Storage
- **Vercel Blob** for meme images and emoji assets.
- Images: max 5MB, auto-compressed to WebP.
